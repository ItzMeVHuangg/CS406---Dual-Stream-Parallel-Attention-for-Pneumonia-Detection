# CS406---Dual-Stream-Parallel-Attention-for-Pneumonia-Detection
Dự án nghiên cứu và phát triển mô hình học sâu cải tiến ứng dụng kiến trúc Dual-Stream kết hợp cơ chế Parallel Attention (PAFM) để tự động phát hiện vùng viêm phổi trên ảnh X-quang ngực. Dự án được thực hiện trên bộ dữ liệu RSNA Pneumonia Detection.
